# BRAMON - API

API da BRAMON.

## Autenticação

Todos os requests deverão conter no mínimo os cabeçalhos mencionados abaixo:

```
Authorization: Bearer XOXOXOXOXOXOXOXO
```

Onde `XOXOXOXOXOXOXOXO` é a chave de API do usuário.
